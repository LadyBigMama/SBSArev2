This page is driven entirely by local files in the repo. To update the map later, edit the zone and label data in data/no-land-map-data.js. No Google embed is required.

How To Edit

  1. Open data/no-land-map-data.js.
  2. Edit the polygon points, route points, and label positions.
  3. Adjust colors and notes in the same file.
  4. When the geometry is finalized, replace the homepage iframe with this local map.
